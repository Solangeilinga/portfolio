# 🌐 Portfolio Personnel - Solange Ilinga

Ce projet est un site web personnel moderne et réactif conçu pour présenter mon parcours, mes compétences en ingénierie et mes projets en data science et développement web.

## 🚀 Technologies Utilisées

* **Frontend** : React.js avec Vite pour un développement rapide.
* **Stylisation** : Tailwind CSS pour un design fluide et personnalisé.
* **Animations** : Framer Motion pour des transitions fluides entre les sections.
* **Icônes** : Lucide React.

## ✨ Fonctionnalités

* **Section Hero** : Introduction avec un message de bienvenue dynamique.
* **À Propos** : Présentation de mon profil d'étudiante en ingénierie passionnée par l'innovation.
* **Compétences** : Visualisation de mes expertises techniques.
* **Projets** : Galerie interactive présentant mes réalisations, notamment dans l'analyse de données et le développement d'applications.
* **Expériences & Certifications** : Détail de mon parcours professionnel et académique.
* **Contact** : Formulaire intégré pour faciliter la communication.

## 📂 Structure du Code Source

* `src/components/` : Contient les composants réutilisables (Header, Footer, ProjectCard, etc.).
* `src/data/projectsData.js` : Fichier centralisant les informations sur mes projets.
* `src/App.jsx` : Point d'entrée principal gérant la structure de la page.
* `public/cv.pdf` : Mon curriculum vitae accessible en téléchargement.

## 🛠️ Installation et Lancement

1.  **Cloner le dépôt** :
    ```bash
    git clone https://github.com/solangeilinga/portfolio.git
    cd portfolio
    ```

2.  **Installer les dépendances** :
    ```bash
    npm install
    ```

3.  **Lancer le serveur de développement** :
    ```bash
    npm run dev
    ```

4.  **Construire pour la production** :
    ```bash
    npm run build
    ```
